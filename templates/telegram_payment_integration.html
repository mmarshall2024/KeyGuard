Telegram Payment Integration

Connect your payment system with Telegram for instant notifications

Group Chat Configuration

Your Telegram group chat ID is configured and ready:

{{ chat_id }}
This ID is used to send payment notifications to your Telegram group

Available Bot Commands

Use these commands in your Telegram group to manage payments:

/payment_status

Check the current status of all payment systems

/revenue_today

Display today's total revenue and transaction count

/payment_methods

Show all 8 available payment methods with setup status

/create_payment_link [amount] [product]

Generate instant payment links for any amount and product

Example: /create_payment_link 297 "OMNI Bot Premium"
/payment_analytics

Show detailed payment analytics and revenue breakdown

Setup Instructions

1. Add Bot to Group

Add your OMNI Empire bot to the Telegram group using chat ID:

Chat ID: {{ chat_id }} Bot Username: @YourOMNIBot
2. Configure Permissions

Give the bot these permissions:

  • Send messages
  • Read messages
  • Add users
  • Pin messages

  • Failed payment alerts
  • High-value transaction alerts
